This is my attempt to graphh out an efficient frontier with an undervisified portfloio with EOD prices fetched from the Yahoo Finance API. I computed expected returns with numpy random to determine weights. This is simply a first attempt at graphing the efficient frontier based on Harry Markowitz's Modern Portfolio Theory. Next steps are to play with a different API called Polygon, use Montecarlo Simulation to generate different allocations, generate a more diversified portfolio, utilize the CAPM startegy to predict expected returns, and compare the present graph with the new one.
